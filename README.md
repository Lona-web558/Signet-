# Signet

Sign PDFs from anywhere — no printer, no scanner, no server.

**Live:** https://lonalonakie.neocities.org/Signet

## Overview

Signet is a single-file, fully client-side PDF signing tool. Upload a document, draw or type your signature, place it on the page, add a date, and download the signed file — all inside the browser tab. Nothing is uploaded to a server; the document never leaves the device.

## Features

- 📄 **Upload & preview** — drag-and-drop or tap to choose a PDF, with page navigation (1/1, ‹ ›)
- 🖋 **Signature placement** — drop your signature anywhere on the page
- 📅 **Date stamping** — add the current date alongside your signature
- ✍️ **Two signature modes**
  - **Draw** — freehand signature via canvas/ink input
  - **Type** — typed signature rendered in a choice of fonts (Caveat, Grotesk, Mono)
- 🖊 **Ink controls** — clear and redraw at any time
- ⬇ **Download** — export the signed PDF directly to device
- 🔒 **Privacy-first** — 100% on-device processing, nothing leaves the browser

## Tech Stack

- HTML5 / CSS3 / vanilla JavaScript
- Canvas API for freehand signature capture
- Client-side PDF rendering and manipulation
- No backend, no database, no external uploads

## Design

Dark terminal aesthetic with JetBrains Mono / Space Grotesk typography and gold/cyan accents, consistent with the rest of the portfolio.

## Status

Live and functional. Single-page, single-file architecture — no build step required.
